# Política de Segurança

## Sobre este repositório

Este repositório contém o portfólio acadêmico e pessoal de Felipe Avila Borges, 
estudante de Ciência da Computação (UniCEUB), hospedado via GitHub Pages.

## Versões suportadas

Como este é um projeto de portfólio pessoal/acadêmico (sem versões de software 
em produção com dependências críticas), apenas a branch `main` é mantida e 
considerada a versão oficial e segura do site.

| Branch | Suportada |
| ------ | --------- |
| main   | ✅ |

## Reportando uma vulnerabilidade

Caso você identifique alguma vulnerabilidade de segurança neste repositório 
(por exemplo, exposição acidental de dados sensíveis, links maliciosos, 
ou problemas de configuração do GitHub Pages), por favor:

1. **Não abra uma Issue pública** descrevendo a vulnerabilidade.
2. Entre em contato diretamente através do e-mail informado no perfil do GitHub 
   do autor, descrevendo:
   - O problema encontrado;
   - Como reproduzi-lo (se aplicável);
   - O impacto potencial.

O autor se compromete a analisar o relato e tomar as medidas necessárias o 
mais rápido possível.

## Boas práticas adotadas neste repositório

- Autenticação em dois fatores (2FA) habilitada na conta;
- Proteção da branch `main`, exigindo Pull Request antes do merge;
- Uso de `.gitignore` para evitar o envio de arquivos sensíveis 
  (credenciais, variáveis de ambiente, chaves de API);
- Repositório público apenas com código-fonte estático (HTML/CSS), 
  sem dados pessoais sensíveis ou credenciais.
